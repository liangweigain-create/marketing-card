# 市场营销与获客指南

建果名片不仅是名片工具，更是企业获客的利器。本指南介绍如何利用数据驱动营销。

## 核心功能

### 数据罗盘

全方位展示名片访问数据：

- **浏览量统计**：追踪每张名片的访问次数
- **访客分析**：了解访客的地区、设备分布
- **行为路径**：分析访客在名片上的浏览轨迹
- **转化追踪**：统计留言、电话点击等转化行为

### 素材库

企业营销素材的统一管理中心：

- 产品介绍 PDF
- 案例展示图片
- 宣传视频
- 报价单模板

### 访客追踪

实时了解谁在看您的名片：

| 信息类型 | 说明                 |
| -------- | -------------------- |
| 访客身份 | 自动识别已知客户     |
| 访问时间 | 精确到分钟的访问记录 |
| 停留时长 | 了解客户关注点       |
| 互动行为 | 是否保存、转发、留言 |

## 使用场景

### 展会获客

1. 展前：准备电子名片和素材
2. 展中：快速交换名片
3. 展后：分析访客数据，精准跟进

### 日常销售

- 通过访客通知及时跟进有意向客户
- 根据客户浏览内容准备针对性话术
